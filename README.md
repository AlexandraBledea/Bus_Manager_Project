# Bus_Manager_Project

# Backend:
- Java
- Spring
- Hibernate
- JPA Repository
- PostgreSQL
- Rest APIs
- CriteriaAPI repositories
- JPQL repositories
- SQL repositories

# Frontend:
- Angular

# Functionalities:

1. Bus component:
- See buses
- Sort buses based on the capacity
- Add new bus
- Update bus
- Delete bus

2. City component:
- See cities
- Add city
- Update city
- Delete city

3. Driver component:
- See drivers
- Add driver
- Update driver
- Delete driver

4. Stations component:
- See stations
- Add station
- Update station
- Delete station

5. Bus stops component:
- See stops
- Add stop
- Update stop
- Delete stop
